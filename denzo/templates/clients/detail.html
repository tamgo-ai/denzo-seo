How to get your GitHub Token
Personal Access Token (Classic)
  1. Go to github.com → click your avatar (top right) → Settings
  2. In the left menu, scroll down to Developer settings (at the bottom)
  3. Click Personal access tokens → Tokens (classic)
  4. Click Generate new token (classic)
  5. Give it a name (e.g. denzo-seo), set expiration, and check scope ✓ repo (full repository access)
  6. Click Generate token → copy the token starting with ghp_
⚠️ Copy the token immediately — GitHub only shows it once.

How to create a WordPress App Password
Requires WordPress 5.6+ with Application Passwords enabled
  1. Go to your WordPress admin panel (/wp-admin)
  2. Click your username (top right) → Edit Profile
  3. Scroll down to the Application Passwords section (at the bottom)
  4. Enter a name (e.g. denzo-seo) and click Add New Application Password
  5. Copy the generated password with format xxxx xxxx xxxx xxxx xxxx xxxx
  6. Use your WordPress username in the WP Username field above
⚠️ If you don't see "Application Passwords", the feature may be disabled — enable it in Plugins or contact your hosting provider.
